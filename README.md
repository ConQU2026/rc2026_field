# RC2026 Gazebo Classic 仿真场地搭建

    根据重邮开源的blender场地文件, 将场地和所有KFS模型导出为.dae格式, 并添加到gazebo classic中

    > 注意目前只加载一个KFS到地图中, 可自行在`worlds/robocon2026.world` 文件中添加其他KFS(都在models目录下)

# 场地模型说明

本项目使用了重庆邮电大学 HXC 战队提供的Robocon 2026比赛场景模型, 为了方便导出进行了部分删除与简化

## 调整说明

- 修正了RC2026场地梅林场地摆放错误
- 将武馆的九宫格边框按照规则添加了黑色边缘
- 删除了广告牌与文字

## KFS色号

- **Red**: Hex `#de0716`
- **Blue**: Hex `#002486`
